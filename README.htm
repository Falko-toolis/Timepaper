Timepaper

Automatic time-based wallpaper changer for Windows — no config files, no UI clutter.

Requirements & Installation

Python 3.10 or later must be installed. Then install the dependencies using one of the two methods below:

  • Double-click install.bat in the Timepaper folder — it runs automatically
  • Or run this command in a terminal:
pip install -r requirements.txt
How it works

Timepaper reads image files from the wallpapers/ folder. Each file is named after the time it should appear. When that time is reached, Timepaper automatically sets it as your desktop wallpaper.

At startup, it immediately applies the wallpaper that should currently be active based on your schedule, then sleeps precisely until the next scheduled change — no constant polling, no CPU waste.

PNG files are automatically converted to high-quality JPEG before being applied. This avoids color degradation that Windows introduces when handling PNGs natively.
Naming your images

Rename each image to HH,MM — use a comma instead of a colon, since Windows does not allow colons in filenames.

FilenameTriggers at
08,00.jpg08:00 (8 AM)
11,00.png11:00 (11 AM)
18,00.jpg18:00 (6 PM)
21,00.png21:00 (9 PM)

Supported formats: .jpg, .jpeg, .png, .bmp — no limit on the number of images.

Running Timepaper

Double-click app.pyw — no console window will appear. The app runs silently in the system tray.

  • Apply now — forces the current wallpaper to apply immediately
  • Quit — stops Timepaper
Launch at startup

To make Timepaper start automatically with Windows:

  • Press Win + R and type shell:startup
  • Drop a shortcut to app.pyw into that folder
Folder structure
Timepaper/ ├── app.pyw ├── install.bat ├── requirements.txt ├── how to use.htm └── wallpapers/ ├── 08,00.jpg ├── 11,00.png ├── 18,00.jpg └── 21,00.png
The .cache/ folder is created automatically and stores converted JPEG copies of your images. You can safely delete it — it will be recreated on next launch.
